# Story: i18n Completion — Switcher + s3.4 Retrofit

## User Story
As a bilingual demo facilitator,
I want to switch between Spanish and English from within the app, and have the `GenerateScenarioForm` render in the active locale,
so that I can present the product in either language without hardcoded text.

## Acceptance Criteria

### Scenario: Locale switcher toggles language
```gherkin
Given the user is on any page of the app
When they click the locale switcher in the header
Then the page content re-renders in the selected language
And the URL switches to the target locale prefix (e.g., /es/... ↔ /en/...)
```

### Scenario: Switcher preserves current route
```gherkin
Given the user is on the /es/scenarios page
When they switch to English
Then they land on /en/scenarios (same path, different locale)
And no navigation error occurs
```

### Scenario: GenerateScenarioForm has no hardcoded strings
```gherkin
Given the generate scenario form is rendered
When I inspect the rendered DOM
Then every user-facing string is translated via useTranslations()
And es.json and en.json both contain all form keys
```

### Scenario: Scenario size minimum is enforced
```gherkin
Given the user opens the generate scenario form
When they inspect the client count input
Then the minimum value is 200
And the helper text reflects the new minimum
```

### Scenario: Dates format per active locale
```gherkin
Given the user is viewing a generated scenario
When the locale is Spanish
Then dates display in es-MX format
When the locale switches to English
Then dates display in en-US format
```

## Examples (Specification by Example)

| Input | Action | Expected Output |
|-------|--------|-----------------|
| User on /es/scenarios, clicks "EN" | Switch locale | URL becomes /en/scenarios, content in English |
| User on /en/clients/42, clicks "ES" | Switch locale | URL becomes /es/clients/42, content in Spanish |
| Form renders with locale=es | Observe strings | All labels, placeholders, errors in Spanish |
| Form renders with locale=en | Observe strings | All labels, placeholders, errors in English |
| Client count input | Inspect min attribute | min=200, helper text says "Mínimo 200, máximo 500" |
| Scenario with dates, locale=es | Observe date format | "24 de julio de 2026" |
| Scenario with dates, locale=en | Observe date format | "July 24, 2026" |
