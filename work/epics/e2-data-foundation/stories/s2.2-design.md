# s2.2 Design — Scenario Management API

---

## Response schemas

```python
class ScenarioSummary(BaseModel):
    id: UUID
    name: str
    sector: Sector
    status: str           # "active" | "inactive"
    client_count: int
    created_at: datetime

class ScenarioDetail(ScenarioSummary):
    seed: int | None
    parameters: dict
    source: str
```

---

## Router structure

```python
router = APIRouter(prefix="/api/scenarios", tags=["scenarios"])

@router.get("", response_model=list[ScenarioSummary])
async def list_scenarios(repo: IScenarioRepository = Depends(get_scenario_repo)):
    return await ListScenarios(repo).execute()

@router.post("", response_model=ScenarioDetail, status_code=201)
async def create_scenario(body: CreateScenarioRequest, repo: ...):
    ...

@router.get("/active", response_model=ScenarioDetail)
async def get_active(repo: ...):
    ...

@router.get("/{scenario_id}", response_model=ScenarioDetail)
async def get_scenario(scenario_id: UUID, repo: ...):
    ...

@router.patch("/{scenario_id}/activate", response_model=ScenarioDetail)
async def activate_scenario(scenario_id: UUID, repo: ...):
    ...
```

Note: `/active` route must be declared **before** `/{scenario_id}` to avoid FastAPI treating "active" as a UUID path parameter.
