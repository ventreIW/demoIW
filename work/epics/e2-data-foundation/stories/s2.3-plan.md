### T3: Manual integration smoke test

> **No code changes — validation only**

**Description:** Run the full stack (`docker compose up` backend + `npm run dev` frontend), verify the acceptance criteria end-to-end in a browser. Validate the route guard, activation flow, and visual state.

**Steps:**
1. Start backend: `docker compose up -d db && cd backend && .venv/bin/uvicorn app.main:app`
2. Start frontend: `cd frontend && npm run dev`
3. Navigate to `/` — should redirect to `/scenarios`
4. Verify 3 seed scenarios render as cards
5. Click "Seleccionar" on one — verify activation, cookie set, visual update
6. Navigate to `/` — should NOT redirect (active scenario exists)
7. Verify responsive layout at 375px viewport

**AC reference:** All MUST and SHOULD criteria

## Execution Order

```
T1 (types + API) → T2 (UI) → T3 (smoke)
```

Sequential: each task depends on the previous. T1 must complete before T2 starts (components import types and API client). T3 must complete after T2 (needs running UI).

No parallelizable tasks — the dependency chain is linear.

| After T2 | `rai gate check gate-lint` | Full project |
| After T2 | `rai gate check gate-format` | Full project |

**Note on `--scope`:** The `scripts/test.sh` currently only supports scoped paths for backend (`backend/tests/...`). Frontend vitest is fast enough that unscoped `gate-tests` is acceptable. The backend tests add negligible overhead (<1s).

## Risks & Mitigations

| Risk | Likelihood | Mitigation |
|------|-----------|------------|
| Backend unreachable during T3 | Medium | Backend already deployed in s2.2; use `docker compose up` |
| Seed scenarios missing from DB | Low | s2.1 migration `0003_seed_scenarios.py` ensures 3 scenarios exist |
| `NEXT_PUBLIC_API_URL` not configured | Medium | Default to `http://localhost:8000` in API client |
| Card component doesn't match design expectations | Low | Reuse existing shadcn/ui Card; design confirmed in gemba walk |
| Middleware conflicts with Next.js 15.3 | Low | Standard `middleware.ts` pattern; Next.js 15 stable |
