**Epic:** E2 — Data Foundation

---

## User story

> As a demo facilitator, I want to see all available scenarios on the entry-point screen and activate one with a single click so that the demo begins from a consistent, known dataset.

---

## Acceptance criteria

- [ ] Navigating to `/scenarios` shows all available scenarios as cards
- [ ] Each card displays: scenario name, sector badge, status (`Activo` / `Sin datos` / `Con datos`), client count
- [ ] Clicking "Seleccionar" on a card activates that scenario (calls `PATCH /{id}/activate`)
- [ ] The active scenario card is visually distinguished (highlighted border or badge)
- [ ] Activating a scenario shows a loading state during the API call and updates the card without full page reload
- [ ] Navigating to `/` or any app route when no scenario is active redirects to `/scenarios`
- [ ] Navigating to `/` when a scenario IS active redirects to the operations panel placeholder (route guard works)
- [ ] "Generar nuevo" and "Subir CSV" buttons are visible but disabled/stubbed (no functionality in this story)
- [ ] Page is responsive (cards stack on mobile)
- [ ] No TypeScript errors, no failing Vitest tests
