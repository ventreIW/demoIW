# s2.1 Scope — Domain Schema & Migrations

## In scope

- `app/domain/entities/` — Pydantic BaseModel classes for all 7 entities
- `app/domain/enums.py` — all project enumerations (Sector, PaymentPattern, ScoreCategory, Channel, Tone, CommunicationStatus, ContactResultType)
- `app/domain/exceptions.py` — EntityNotFoundError, InvalidOperationError
- `app/adapters/persistence/models.py` — SQLAlchemy 2.0 ORM models (DeclarativeBase, mapped_column)
- `app/adapters/persistence/mappers.py` — bidirectional ORM ↔ Pydantic conversion functions
- `alembic/versions/0002_domain_schema.py` — creates all 7 tables, FKs, unique index
- `alembic/versions/0003_seed_scenarios.py` — inserts 3 blank scenarios
- ADR-002 in `dev/decisions/`

## Out of scope

- Repository implementations (s2.2)
- Any API endpoints
- Any frontend changes
