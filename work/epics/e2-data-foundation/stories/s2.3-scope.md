# s2.3 Scope — Scenario Selector UI

## In scope

- `src/app/scenarios/page.tsx` — scenario selector page
- `src/components/scenarios/ScenarioCard.tsx` — individual scenario card
- `src/components/scenarios/ScenarioGrid.tsx` — responsive card grid
- `src/lib/api/scenarios.ts` — typed API client functions (fetch wrappers for scenario endpoints)
- `src/types/scenario.ts` — TypeScript types matching backend Pydantic schemas
- Route guard: `src/middleware.ts` (Next.js middleware) — redirect to `/scenarios` if no active scenario cookie/state
- Vitest tests: `ScenarioCard.test.tsx` — renders name, badge, active state correctly
- Backend URL configured via `NEXT_PUBLIC_API_URL` environment variable

## Out of scope

- CSV file picker UI (s2.4)
- "Generar nuevo" form (E3)
- Actual operations or executive panel pages
- i18n (B-15, later epic) — use Spanish strings directly for now
- Authentication
