# s2.4 Scope — CSV Upload

## In scope

- `app/application/use_cases/manage_scenario.py` — `UploadCSVScenario` use case
- `app/adapters/dataset/csv_loader.py` — `IDatasetPort` CSV implementation (validates + parses)
- `POST /api/scenarios/upload-csv` router endpoint
- CSV format documented in `README.md` (required columns, date format)
- Frontend: file input component wired to the upload endpoint
- `tests/test_csv_upload.py` — all error + happy-path cases

## Out of scope

- Scoring or prioritization of the uploaded data (E4)
- Any transformation or enrichment of the CSV data (raw import only)
- Support for Excel files (CSV only per backlog)
