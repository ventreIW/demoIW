# s2.4 Design — CSV Upload

---

## CSV format

Required columns (order-independent, header row required):

| Column | Type | Notes |
|---|---|---|
| `client_name` | string | Synthetic company name |
| `invoice_id` | string | Unique per file |
| `amount` | decimal | Positive number |
| `due_date` | date | ISO 8601 (YYYY-MM-DD) |

Optional columns (imported if present, ignored if absent):
- `sector`, `days_overdue`, `payment_pattern`

## Validation flow (csv_loader.py)

```python
class CSVLoader:
    REQUIRED_COLUMNS = {"client_name", "invoice_id", "amount", "due_date"}

    def load(self, file_bytes: bytes) -> RawDataset:
        df = pd.read_csv(io.BytesIO(file_bytes))
        missing = self.REQUIRED_COLUMNS - set(df.columns)
        if missing:
            raise CSVValidationError(f"Missing columns: {missing}")
        if len(df) == 0:
            raise CSVValidationError("File contains no data rows")
        errors = self._validate_rows(df)
        if errors:
            raise CSVValidationError(errors)  # includes row number + column
        return self._to_raw_dataset(df)
```

## Router endpoint

```python
@router.post("/upload-csv", response_model=ScenarioDetail, status_code=201)
async def upload_csv(
    file: UploadFile = File(...),
    repo: IScenarioRepository = Depends(get_scenario_repo),
    dataset_port: IDatasetPort = Depends(get_dataset_port),
):
    if file.size and file.size > 5 * 1024 * 1024:
        raise HTTPException(413, "File exceeds 5 MB limit")
    content = await file.read()
    scenario = await UploadCSVScenario(repo, dataset_port).execute(content)
    return scenario
```
