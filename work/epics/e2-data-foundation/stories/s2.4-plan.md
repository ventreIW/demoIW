## Risks & Mitigations

| Risk | Likelihood | Mitigation |
|------|-----------|------------|
| CSV dialect edge cases (quoted fields, different delimiters) | Low | `csv.DictReader` handles standard CSV dialects. E3 will address complex CSV needs. |
| Backend not running during TC-2 implementation | High | TC-2 tests mock `uploadCsv` — no backend needed for unit tests. Manual integration (TC-3) requires backend. |
| Server component file input limitation | Medium | `CsvUpload` is a client component. Use Next.js `dynamic(() => import(...), { ssr: false })` if hydration issues arise. |
