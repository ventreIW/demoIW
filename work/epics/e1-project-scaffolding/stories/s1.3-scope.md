# s1.3 Scope — Backend Skeleton

## In scope

- `app/main.py` — FastAPI app factory, CORS middleware, lifespan context
- `app/config.py` — Pydantic Settings v2: DATABASE_URL, APP_ENV, LOG_LEVEL
- `app/infrastructure/database.py` — async SQLAlchemy 2.0 engine, `AsyncSession`, `get_session` dependency
- `app/routers/health.py` — `GET /health` with DB connectivity check
- `alembic/env.py` — configured for async engine + imports ORM models
- `alembic/versions/` — one init migration (empty schema, no tables yet)
- `alembic.ini`
- `.env.example` — DATABASE_URL, APP_ENV, LOG_LEVEL (no OpenRouter key yet — that's E3)
- `tests/conftest.py` — async test client fixture, test DB URL override (SQLite in-memory for CI)
- `tests/test_health.py` — GET /health happy path + DB unavailable path
- `app/infrastructure/logging.py` — structlog JSON setup

## Out of scope

- Any domain models or database tables (E2)
- OpenRouter / LLM configuration (E3)
- Authentication
- Request ID middleware (deferred to cross-cutting story in a later epic)
