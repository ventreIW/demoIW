## User story

> As a developer, I want a working Next.js 15 frontend with layout shell and health check so that all frontend stories have a base application to build into.

---

## Acceptance criteria

- [ ] `cd frontend && npm run dev` starts on localhost:3000 without errors
- [ ] `GET /healthcheck` route returns `{ "status": "ok" }` (JSON)
- [ ] Root page (`/`) renders the layout shell: visible sidebar placeholder and main content area
- [ ] Layout shell is responsive (sidebar collapses on mobile)
- [ ] shadcn/ui is initialized; at minimum `Button`, `Card`, `Separator`, `Skeleton` components are available in `src/components/ui/`
- [ ] `npm run typecheck` exits 0
- [ ] `npm run lint` exits 0
- [ ] At least one Vitest component test passes (healthcheck route test)
- [ ] No `console.error` in the browser on page load
