## In Scope
- Create manifest.json with app name, icons, theme/background colors, and standalone display mode
- Register basic service worker to make the app installable
- Ensure Lighthouse PWA installability checks pass

## Out of Scope
- Push notifications, background sync, advanced caching strategies
- Any configuration UI forms
- Offline page fallbacks
- PWA settings panel

## Done when
- App appears installable in Chrome/Edge browser
- Valid manifest.json is present in the app root
- Basic service worker is registered and functional
- Lighthouse PWA score is >= 90
