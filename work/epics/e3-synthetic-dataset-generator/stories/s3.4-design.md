## Persistence flow

```python
# Updated GenerateDataset use case
async def execute(self, params: GenerationParams, model: str) -> Scenario:
    # 1. Generate procedurally
    raw = self._dataset_port.generate_procedural(params)
    # 2. Enrich with LLM
    enriched = await self._enrichment_service.enrich(raw, model)
    # 3. Create scenario shell
    scenario = Scenario(name=f"{params.sector.value.title()} — {params.seed}", ...)
    await self._scenario_repo.save(scenario)
    # 4. Persist clients, invoices, payments
    clients = [map_raw_client(row, scenario.id) for _, row in enriched.clients.iterrows()]
    await self._client_repo.save_many(clients)
    # invoices and payments similarly
    return scenario
```

Batch insert via SQLAlchemy `insert()` bulk — do not loop `session.add()` for large datasets.

## API endpoint

```python
@router.post("/generate", response_model=ScenarioDetail, status_code=201)
async def generate_scenario(
    body: GenerationParams,
    use_case: GenerateDataset = Depends(get_generate_dataset_use_case),
):
    scenario = await use_case.execute(body, model=settings.model_data_enrichment)
    return scenario_to_response(scenario)
```
