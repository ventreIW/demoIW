## LLMEnrichmentService

```python
class LLMEnrichmentService:
    BATCH_SIZE = 20

    def __init__(self, llm_port: ILLMPort, prompt_dir: Path):
        self._llm = llm_port
        self._template = (prompt_dir / "data_enrichment" / "v1_company_description.txt").read_text()

    async def enrich(self, raw_dataset: RawDataset, model: str) -> RawDataset:
        clients = raw_dataset.clients.copy()
        batches = [clients.iloc[i:i+self.BATCH_SIZE] for i in range(0, len(clients), self.BATCH_SIZE)]
        enriched_rows = []
        for batch in batches:
            try:
                enriched = await self._enrich_batch(batch, model)
                enriched_rows.extend(enriched)
            except ExternalServiceError:
                # graceful degradation: keep original Faker names
                enriched_rows.extend(batch.to_dict("records"))
        clients = pd.DataFrame(enriched_rows)
        return RawDataset(clients=clients, invoices=raw_dataset.invoices, payments=raw_dataset.payments)
```

## Prompt template `v1_company_description.txt`

```
You are generating fictional company data for a financial software demonstration.
Generate names and descriptions for {count} companies in the {sector} sector operating in Mexico.

Rules:
- Names must be completely fictional — do not use real company names.
- Descriptions must be one sentence, professional, and sector-appropriate.
- Output ONLY a JSON array with no additional text:
  [{"name": "...", "description": "..."}, ...]

Sector: {sector}
Count: {count}
```

## GenerateDataset use case

```python
class GenerateDataset:
    def __init__(
        self,
        dataset_port: IDatasetPort,
        enrichment_service: LLMEnrichmentService,
    ):
        ...

    async def execute(self, params: GenerationParams, model: str) -> RawDataset:
        raw = self.dataset_port.generate_procedural(params)
        enriched = await self.enrichment_service.enrich(raw, model)
        return enriched
```
